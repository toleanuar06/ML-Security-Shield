## ⚙️ Қалай іске қосуға болады? (Орнату нұсқаулығы)

Бұл жоба екі бөліктен (Python сервері және Chrome кеңейтімі) тұрады. Жүйені өз компьютеріңізде іске қосу үшін төмендегі қадамдарды ретімен орындаңыз.

### 📌 Алдын ала қойылатын талаптар
Компьютеріңізде мына бағдарламалар орнатылғанына көз жеткізіңіз:
* **Python** (3.8 немесе одан жоғары нұсқасы)
* **Google Chrome** браузері
* **Git** (кодты жүктеп алу үшін)

---

### 1-қадам: Жобаны жүктеп алу
Алдымен репозиторийді өз компьютеріңізге көшіріп алыңыз. Терминалды (немесе Command Prompt) ашып, мына бұйрықты жазыңыз:
```bash
git clone https://github.com/СІЗДІҢ_ЛОГИНІҢІЗ/ML-Security-Shield.git
cd ML-Security-Shield
```
*(Ескерту: `СІЗДІҢ_ЛОГИНІҢІЗ` деген жерге өз GitHub логиніңізді жазыңыз).*

### 2-қадам: Виртуалды ортаны құру (Міндетті емес, бірақ ұсынылады)
Кітапханалар басқа жобалармен араласып кетпеуі үшін оқшауланған орта құрамыз:

**Windows үшін:**
```bash
python -m venv .venv
.venv\Scripts\activate
```

**Mac/Linux үшін:**
```bash
python3 -m venv .venv
source .venv/bin/activate
```

### 3-қадам: Қажетті кітапханаларды орнату
Машиналық оқыту және сервер жұмыс істеуі үшін қажетті пакеттерді жүктейміз:
```bash
pip install fastapi uvicorn pandas scikit-learn joblib pydantic
```

### 4-қадам: Серверді іске қосу (Backend)
Барлығы орнатылған соң, жүйенің "миын" (FastAPI серверін) қосамыз. Терминалға мына бұйрықты енгізіңіз:
```bash
uvicorn api:app --reload
```
✅ Егер терминалда `INFO: Uvicorn running on http://127.0.0.1:8000` деген жазу шықса, сервер сәтті қосылды деген сөз! **Терминалды жаппаңыз**, ол артқы фонда жұмыс істеп тұруы керек.

---

### 5-қадам: Chrome кеңейтімін орнату (Frontend)
Сервер қосылып тұрған кезде, браузердегі плагинді іске қосамыз:
1. **Google Chrome** браузерін ашыңыз.
2. Жоғарғы іздеу жолағына мына мекенжайды жазып, Enter басыңыз: `chrome://extensions/`
3. Ашылған терезенің оң жақ жоғарғы бұрышындағы **"Developer mode"** (Әзірлеуші режимі / Режим разработчика) қосқышын қосыңыз.
4. Сол жақ жоғарыда пайда болған **"Load unpacked"** (Загрузить распакованное расширение) батырмасын басыңыз.
5. Пайда болған терезеден біздің жобаның ішіндегі **`ML_Extension`** папкасын тауып, соны таңдаңыз.
6. Кеңейтімдер тізімінде "ML Security Shield" пайда болуы керек.
