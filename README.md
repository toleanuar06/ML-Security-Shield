# 🛡️ ML Security Shield (Зияткерлік Киберқауіпсіздік Жүйесі)

**Машиналық оқыту алгоритмдері мен браузер кеңейтімі арқылы қауіпті (фишингтік) сайттарды нақты уақытта анықтап, бұғаттайтын зияткерлік жүйе.**

👤 **Авторы:** Ануар Төле (және өз тобыңыздағы адамдардың атын осында қосыңыз)

---

## 📌 Жоба туралы
Қазіргі таңда киберқылмыскерлер күн сайын мыңдаған жаңа фишингтік сайттар ашады. Дәстүрлі антивирустар олардың бәрін «қара тізімге» енгізіп үлгермейді. 
**ML Security Shield** — сайттың атына емес, оның **құрылымы мен ниетіне** қарайтын ақылды жүйе. Ол кез келген сілтемені (URL) ашылмас бұрын 10 түрлі математикалық белгі бойынша сараптап, 1-2 миллисекунд ішінде қауіпті бұғаттайды.

## 🚀 Жүйенің басты артықшылықтары
- **Ауқымды деректер базасы:** Жасанды интеллект моделі (Random Forest) **450 000-нан астам** нақты веб-сілтемелермен оқытылған.
- **Жылдамдық:** Тексеру процесі өте жылдам (1-2 мс) өтеді және браузердің жұмысына мүлдем кедергі келтірмейді.
- **Терең талдау:** Сілтемелер ұзындығы, күдікті сөздер, IP-адрестердің жасырылуы, арнайы таңбалар жиілігі сияқты 10 түрлі параметр (Feature Extraction) бойынша талданады.
- **Екіжақты архитектура:** Жеңіл Chrome кеңейтімі (Frontend) және қуатты FastAPI Python сервері (Backend).
